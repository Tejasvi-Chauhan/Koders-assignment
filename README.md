# Task Manager Web Application

A simple full-stack Task Management Web Application.

##  Features

- Create a new task (Title + Description)
- View all tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as Completed or Pending
- Basic form validation
- Proper error handling

---

##  Tech Stack

### Frontend
- React.js (or HTML/CSS/JS if used)
- Axios (for API calls)
- DaisyUI / Tailwind (if used)

### Backend
- Node.js
- Express.js

### Database
- MongoDB
